Server Requirements

  • PHP >= 7.3
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • GD Library PHP Extension

*In most servers, these extensions are enabled by default, but you should check with your hosting provider.

Installation

#Step 01

  1. Extract the downloaded project zip file on your PC.
  2. Upload the wallpaper71.zip file to your server public_html or any other directory you intend to run the script.
  3. Extract the zip file in that directory.
  4. Then create a new database from your server MySQL database.
  5. Create a DB user to the database and link that database to the DB user.

#Step 02

  1. First, hit your site URL and it will automatically take you to the installation.
  2. You will get the required information. If everything is ok then click on next step.
  3. Now you need to set Database Host, Database Name, Database Username, Database Password, and click Continue.
  4. Now fill up the information as you want and click Continue.
  5. After that login page will appear, now you can login with your given e-mail address and password.

Admin Configurations

#Step 01 - Add Categories

  1. Go to categories tab from left sidebar and click add new category to add category
  2. Add category name, Add category description, Choose Cover photo (Image size should be 720px X 320px) and click on Submit button

#Step 02 - Add Wallpapers

  1. Go to wallpapers tab from left sidebar and click add new wallpaper to add wallpaper
  2. Choose your photo, Add wallpaper name, Choose wallpaper category, Add wallpaper tag, Add wallpaper description and click on Submit button

#Step 03 - Add Colors

  1. Go to colors tab from left sidebar and click add new color to add color
  2. Add color name, Choose color photo (Image size should be 300px X 200px) and click on Submit button

#Step 04 - App Settings

  1. Go to app settings tab from left sidebar
  2. Add app title, Add About us Link, Add About us Link, Choose admin logo, Choose admin favicon, Choose Most Popular Item Background Image, Choose Recent Photo Item Background Image and click on Submit button

#Step 05 - Ads & Notification

  1. Go to ads & notification tab from left sidebar
  2. On Ads Status, Add AdMob App Open Ad Unit ID, Enable AdMob Banner Ad Unit ID, Add AdMob Banner Ad Unit ID, Enable AdMob Interstitial Ad Unit ID, Add AdMob Interstitial Ad Unit ID, Add Push notification authorization key and click on Submit button

#Step 06 - Update Profile

  1. Go to update profile tab from left sidebar
  2. Update your name, Update your email, Your carrent password, Add your new password, Add your confirm password and click on Update button

#Step 07 - Push Notification

  1. Go to console.firebase.google.com and click to Create project
    https://console.firebase.google.com/
  2. Type your project name and click continue
  3. Click continue to proceed to the next step
  4. Accept terms and click create project
  5. Click over the android icon to proceed to the next step
  6. Add your android package name, Add your app nickname and click register app
  7. Download google.services.json file and click next to proceed to the next step
  8. Click next to complete the procedure
  9. Click over your app name (wallpaper) and click over to the icon to manage settings
  10. Click to cloud messaging tab and copy the server key
  11. Go to app's admin and select Ads & Notification tab from left sidebar and paste copied key to Push notification authorization key field and click to submit
  12. open source file and select gradle from right sidebar, select wallpaper71 > tasks > signingReport , copy SHA1 code from below console bar
  13. Go back to your app console's setting page and select general tab, select add fingerprint file, paste copied SHA1 code and click on Save button
  14. Go to app's admin and select send push notification tab from left sidebar and click to new notification to send notification
  15. Add Notification title, Add Notification descprition, choose collection and click on Send button

Common Errors

#1 - Extension (admin)

If you don't get here all options checked then please contact your service provider.

#2 - Network-security-config (app)

If your data is not loading please check the step 09 "Network Security Permission" and make sure you complete the step.

#3 - Subfolder admin install note

If you installed your admin panel in a subfolder then please add "subfolder name/public" to the URL. Like "xyz.com/xyz/public"

#4 - Base Url

If your data is not loading and you installed the admin panel in a subfolder please make sure you "Set Your Base Url" like "domain.com/subfolder/public".

#5 - When Admin Settings Change then App Re-Open

If you have changed admin settings then please close the app and re-open it to view the change.

#6 - Wallpaper upload error

If you are getting an error at the time of uploading wallpaper please go to cpanel, search for INI, place your URL, increase "post_max_size" and "upload_max_filesize", click to "apply" button.
